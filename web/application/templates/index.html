About Nekobot

Nekobot is a powerful Discord bot that serves users with a clean solution to add extra functionality to their Discord server. Whether you are pulling thousands of cat gifs from Giphy or whacking away the ban hammer on some hecklers in your Discord server, Nekobot is there help!

We feature a few plugins like the Nekobot collection that aids in the creation and moderation of a Nekobot supported Discord server. The Moderator plugin allows basic moderator functions like !kick and !ban. The Giphy plugin interprets two commands, !gif and !giphy, by taking into a keyword string that then queries the Giphy collection against the given keyword string and returns a random gif from the results back to Discord. We're currently working on furthering the modularity of our plugin framework so people like you can contribute to Nekobot! Check out our Github for a contribution guide!
